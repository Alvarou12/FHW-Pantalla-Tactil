# FHW-Pantalla-Tactil

1. Definición del Periférico

Una pantalla táctil (o touch screen) es un dispositivo de entrada y salida (E/S) que se coloca sobre la pantalla de un dispositivo informático (como un monitor, tablet o smartphone). Permite al usuario interactuar con el contenido visible en pantalla tocando directamente su superficie.

Se clasifica como un periférico dual, ya que:

    Salida: Muestra la información visual (como un monitor estándar).

    Entrada: Acepta comandos mediante la presión o el contacto, sustituyendo al ratón o al teclado para ciertas funciones.

2. Características Principales (Breves)

    Interfaz Directa: Permite la interacción directa sin dispositivos intermedios.

    Multitáctil (Multitouch): La mayoría de las modernas soportan múltiples puntos de contacto simultáneos (p. ej., gestos de pellizco para zoom).

    Resolución: Capacidad para mostrar imágenes detalladas, medida en píxeles.

    Sensibilidad: Depende de la tecnología; puede ser sensible a la presión (resistiva) o a la capacitancia (capacitiva).
